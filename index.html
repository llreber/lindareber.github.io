Weather Data

The purpose of this project was to examine the changes in weather versus distance from the equator. 500 cities were randomly selected and weather data was obtained using the OpenWeatherMap API.

Using the API, weather data was obtained on September 26, 2018. Using Matplotlib, temperature, percent cloud cover, humidity, and wind speed were plotted versus latitide. This site provides the results of that analysis.
